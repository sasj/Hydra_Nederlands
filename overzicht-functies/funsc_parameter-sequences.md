### Lijst als parameter reeksen

```javascript
osc(
   [80, 100, 200, 50], 1 )
)
.out(o0)
```

### Functies voor parameter reeksen

#### snelheid - fast

`fast ( hoeveelheid ) `

* `hoeveelheid` :: float (default `x`)

```javascript
osc(
   [80, 100, 200, 50].fast(0.2), 1 )
)
.out()
```
