## webcam

`s0` = de S van source oftewel bron
`initCam()` = functie dat het webcam beeld pakt


```javascript
s0.initCam() // webcam naar s0

src(s0)
.out()
```
het kan zijn dat je een pop-up krijgt om gebruik van de camera toe te laten.
